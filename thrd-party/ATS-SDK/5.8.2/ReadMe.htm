Introduction

Welcome to the AlazarTech SDK.

 

This SDK provides the files and documentation necessary for programmers to use functions exported by AlazarTech device drivers to acquire data from AlazarTech PCI/PCIe digitizer boards. It includes: a programmer’s guide and reference manual, header and library files, and sample code.

Getting started

A user manual is shipped with each AlazarTech digitizer board. This manual describes the features of your digitizer board, and provides its specifications. We recommend that you read the manual, and become familiar with the board’s features. This will make the task of writing software to control the hardware much easier.

 

The “AlazarTech SDK Guide” is included with this SDK. This document provides:

·        A programmer’s guide that describes how to configure, and acquire data from, digitizer boards.

·        A reference manual that describes the functions exported by the device drivers.

 

Please read the programmer’s guide section of the “AlazarTech SDK Guide”. It should provide the instructions necessary to use the AlazarTech API to acquire data from AlazarTech digitizer boards.

Sample programs

The AlazarTech SDK includes C/C++ and C# sample programs that demonstrate many of the different types of acquisitions possible with AlazarTech digitizer boards. The samples are organized into folders under the “Samples” and “Samples_CSharp” folders. There is one folder per board model.
